# Steushio Cenverter (FFmpeg Studio)

A fully standalone, native desktop application for high-performance FFmpeg video and audio conversion. 
Built with Python and CustomTkinter, it features a beautiful dark-mode interface, drag-and-drop batch processing, and live conversion tracking.

## Features
- **Drag & Drop Batch Processing**: Seamlessly queue multiple media files.
- **Smart Presets & Toggles**: Pre-configured pipelines for Extract Audio, Remux, OBS Fix, Pro-Editing targets, and compression.
- **Granular FFmpeg Control**: Easily adjust audio/video Codecs, Bitrates, CRF, and Channels.
- **Standalone Portable EXE**: Bundled directly with FFmpeg via PyInstaller, requiring zero installation or Python dependencies on the host machine.
- **Live Terminal Logging**: Visual progress bars mapped directly from the FFmpeg background `subprocess`.
